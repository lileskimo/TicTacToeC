
Tic Tac Toe
Functionality:
You can play Tic Tac Toe against the computer.
The program allows you to choose 'X' or 'O' as your symbol.
The game will display the board and prompt you to make your move.
After each move, the computer will make its move automatically.
The game ends when either you or the computer wins, or if it's a draw.
You can choose to play again or exit the game.

Functions Used:

choose: Allows the player to choose their symbol ('X' or 'O').
displayboard: Displays the current state of the Tic Tac Toe board.
gameOn: Checks if there are any moves remaining on the board.
evaluate: Evaluates the current state of the board to determine if there is a winner.
minimax: Implements the minimax algorithm to determine the best move for the computer player.
findBestMove: Finds the best move for the computer player using the minimax algorithm.
playtictactoe: Manages the gameplay loop for Tic Tac Toe.

Program Execution and Flow:
1. Upon running the program, you are presented with a main menu.
2. Choose option number 1 for Tic Tac Toe.
3. You will be prompted to choose 'X' or 'O' as your symbol.
4. The game board will be displayed.
5. Make your move by entering the row (1-3) and column (1-3) for your move.
6. The computer will make its move automatically.
7. Steps 5-6 will repeat until the game ends.
8. If you win, lose, or the game ends in a draw, the result will be displayed.
9. You will be asked if you want to play again.
10. If you choose to play again, the game will reset.
11. If you choose to exit, the program will terminate.
